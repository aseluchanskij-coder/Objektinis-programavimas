# Objektinis-programavimas
Ši programa nuskaito studento vardą ir pavardę bei namų darbų ir egzamino pažymius.
Duomenys saugomi struktūroje, o galutinis balas skaičiuojamas naudojant vidurkį arba medianą.
Vartotojas gali pasirinkti skaičiavimo metodą – Galutinis (Vid.) arba Galutinis (Med.).
Rezultatai išvedami ekrane dviejų skaičių po kablelio tikslumu.
